# 文档转PDF转换工具

一个Windows专用的Python工具，批量将Excel和Word文档转换为黑白PDF。支持页面提取、自动依赖安装和错误日志记录。

## 快速开始

```bash
pip install pywin32 pypdf
python doc2pdf.py
```

## 功能

- Excel工作表1转PDF
- Word文档转PDF  
- 基于关键词的页面提取
- 黑白PDF输出
- 批量处理

## 要求

- Windows系统
- Python 3.7+
- Microsoft Office
